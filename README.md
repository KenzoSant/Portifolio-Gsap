## 📋 Sobre

Este projeto é um portfólio pessoal desenvolvido para apresentar projetos, habilidades e experiências como desenvolvedor front-end. A aplicação foi construída com foco em performance, design moderno e interações avançadas, proporcionando uma experiência imersiva ao usuário.

---

## ✨ Funcionalidades

### 🎨 Interface e Experiência

* ✅ Design moderno e responsivo (mobile-first)
* ✅ Interface interativa com foco em UX
* ✅ Navegação fluida entre seções

### ⚡ Animações e Interações

* ✅ Animações com GSAP (GreenSock)
* ✅ Scroll suave com Lenis
* ✅ Efeito parallax em elementos
* ✅ Cursor customizado
* ✅ Botões com efeitos interativos
* ✅ Animações de texto com SplitType

### 📬 Funcionalidades

* ✅ Formulário de contato funcional com EmailJS
* ✅ Seção de projetos dinâmica
* ✅ Seção de skills interativa

---

## 🛠️ Tech Stack

### Front-End

* **HTML5** - Estrutura semântica
* **CSS3** - Estilização e responsividade
* **JavaScript (ES6+)** - Lógica e interatividade

### Bibliotecas

* **GSAP** - Animações avançadas
* **Lenis** - Scroll suave
* **SplitType** - Animações de texto

### Ferramentas

* **EmailJS** - Envio de formulários
